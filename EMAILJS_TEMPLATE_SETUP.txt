EMAILJS BOOKING FORM - COMPLETE TEMPLATE VARIABLES
====================================================

Copy and paste these variables into your EmailJS template:

═══════════════════════════════════════════════════════════════
ALL FORM VARIABLES BEING SENT
═══════════════════════════════════════════════════════════════

1. guest_information          - Contains: Name + Email + Phone (combined)
2. check_in_date              - Check-in date (e.g., 2026-04-15)
3. check_out_date             - Check-out date (e.g., 2026-04-20)
4. number_of_guests           - Number of guests (1-10)
5. message                    - Special requests/message
6. property_name              - Property name (auto-filled)
7. property_price             - Price per night (auto-filled)
8. property_location          - Location/arrondissement (auto-filled)
9. property_registration      - Registration number (auto-filled)
10. card_number               - Last 4 card digits (masked: ****1234)
11. card_expiration           - Expiration date (MM/YY format)
12. card_cvv                  - CVV (masked as *** for security)
13. billing_street            - Street address
14. billing_apartment         - Apartment/suite number
15. billing_city              - City name
16. billing_state             - State/province
17. billing_zip               - ZIP/postal code
18. billing_country           - Country (dropdown selection)
19. to_email                  - Recipient email: erentalscaddex@gmail.com
